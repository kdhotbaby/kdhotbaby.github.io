Cobra

Cobra 可视化Python虚拟项目。

就像Java和C#一样，Python是基于栈的虚拟机。如果能够“看到”Python虚拟机执行的过程将是一个非常有趣的。Cobra就是这样一个项目。在Cobra中，你会看到一个Python语句中有多少字节码会被执行，字节码如何影响命名空间和虚拟机的栈帧。理解虚拟机的机制会提高你对Python语言的理解。

Corbra主要由三个组件组成：VM虚拟机，Web服务器和Web客户端。Web服务器使用Django实现。Web客户端通过Ajax与Web服务器通信。Web服务器在收到客户端的命令后，通过socket连接到VM服务器获取信息。VM服务器是修改后的Python。

Browser <—- ajax —-> Web Server(Django) <—- socket —-> VM Server(Modified Pythonn)
